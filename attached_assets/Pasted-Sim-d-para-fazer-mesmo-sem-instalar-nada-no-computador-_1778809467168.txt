Sim, dá para fazer mesmo sem instalar nada no computador 😊
Como o seu GitHub/Codespaces não está funcionando, a melhor saída é usar um ambiente online gratuito para rodar o NestJS.

As opções mais fáceis hoje são:

* [StackBlitz](https://stackblitz.com?utm_source=chatgpt.com)
* [Replit](https://replit.com?utm_source=chatgpt.com)
* [GitPod](https://gitpod.io?utm_source=chatgpt.com)

Para o seu caso de ADS com NestJS + MongoDB, eu recomendo mais o **Replit**, porque ele já abre terminal, instala dependências e deixa rodar API sem precisar configurar muita coisa.

---

# O caminho mais fácil (sem instalar nada)

## 1. Criar conta no Replit

Entre em:

[Replit](https://replit.com?utm_source=chatgpt.com)

Pode entrar com Google mesmo.

---

# 2. Criar projeto Node.js

Depois:

* Clique em **Create Repl**
* Escolha **Node.js**
* Nome: `proj_aula7`

---

# 3. Abrir o terminal

No Replit existe um terminal igual ao VSCode.

Digite:

```bash
npm install -g @nestjs/cli
```

Depois:

```bash
nest new .
```

IMPORTANTE:
Quando perguntar sobre package manager, escolha:

```bash
npm
```

---

# 4. Rodar a aplicação

Depois que terminar:

```bash
npm run start
```

Se funcionar, vai aparecer algo tipo:

```bash
[Nest] application successfully started
```

O Replit gera um link público automaticamente.

---

# 5. Testar o “Hello World”

Abra a aba “Webview” ou clique no link que o Replit mostrar.

Você deve ver:

```text
Hello World!
```

---

# 6. Criar os arquivos pedidos

No terminal:

```bash
nest g module exemplo
nest g controller exemplo
nest g service exemplo
```

Depois crie manualmente:

```text
src/exemplo/exemplo.model.ts
```

---

# 7. MongoDB sem instalar nada

Você também NÃO precisa instalar MongoDB.

Use o MongoDB Atlas gratuito:

[MongoDB Atlas](https://www.mongodb.com/cloud/atlas/register?utm_source=chatgpt.com)

Plano free funciona perfeitamente para atividade da faculdade.

---

# 8. Instalar dependências Mongo

No terminal:

```bash
npm install @nestjs/mongoose mongoose
```

---

# 9. Configurar o app.module.ts

Depois você adiciona algo assim:

```ts
import { Module } from '@nestjs/common';
import { MongooseModule } from '@nestjs/mongoose';

@Module({
  imports: [
    MongooseModule.forRoot('SUA_STRING_DO_MONGODB'),
  ],
})
export class AppModule {}
```

---

# 10. CORS (se precisar)

No `main.ts`:

```ts
app.enableCors({
  origin: '*',
  methods: 'GET,HEAD,PUT,PATCH,POST,DELETE,OPTIONS',
  credentials: true,
});
```

---
